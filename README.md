# Kacey 微记

每日微日记工具，帮助记录生活、捕捉灵感、管理选题。

## 功能

- **日记** — 每天记录亮点、收获和明日目标，支持连续打卡统计
- **灵感速记** — 随时记下选题、台词、画面、BGM 等灵感
- **选题看板** — 拖拽式看板管理「想拍 → 在拍 → 已发」
- **文案生成** — 一键生成小红书/微博/抖音/知乎风格文案，支持多语言视频脚本
- **导出** — 支持导出为 JSON 和 Markdown

## 使用方式

打开 `index.html` 即可使用，也可以用本地服务器：

```bash
python serve.py
# 然后访问 http://localhost:3456
```

数据保存在浏览器的 localStorage 中。
